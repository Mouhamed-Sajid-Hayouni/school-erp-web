# Demo Script

## Admin Flow
1. Login as admin
2. Open Users
3. Create a teacher or student
4. Open Classes
5. Open Subjects
6. Open Schedules
7. Edit a schedule
8. Open Attendance
9. Save attendance
10. Open Grades
11. Save grades

## Student Flow
1. Logout
2. Login as student
3. Show timetable
4. Show grades
5. Show absences

## Parent Flow
1. Logout
2. Login as parent
3. Show linked child
4. Show child timetable
5. Show child grades
6. Show child attendance

## Mobile Flow
1. Open Android APK
2. Login as student
3. Show student portal
4. Logout
5. Login as parent
6. Show parent portal
